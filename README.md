## Development

### Running Locally

```bash
# Install dependencies
bundle install

# Start local server
bundle exec jekyll serve

# View at http://localhost:4000
```

### Adding New Chapters

```bash
# Create a new chapter (numbering determines order)
./scripts/new_chapter.sh 01-new-chapter "Chapter Title"
```
